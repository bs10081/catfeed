# 滅霸的餵食記錄系統

這是一個用於記錄流浪貓滅霸（默默、雞腿）餵食情況的網頁應用。

## 功能特點

- 記錄餵食的食物類型和份量
- 支持添加備註信息
- 查看歷史餵食記錄
- 響應式設計，支持手機訪問

## 安裝步驟

1. 安裝依賴包：
```bash
pip install -r requirements.txt
```

2. 運行應用：
```bash
python app.py
```

3. 打開瀏覽器訪問：
```
http://localhost:5000
```

## 技術棧

- 後端：Flask
- 數據庫：SQLite
- 前端：Bootstrap 5
- CSS：自定義樣式
